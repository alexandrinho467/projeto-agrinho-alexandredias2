# 🌾 Herança de Terra e Futuro — Projeto Agrinho

Repositório oficial do projeto web desenvolvido para o **Concurso Agrinho do Paraná**, voltado para alunos do Ensino Médio. A aplicação aborda o tema *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*.

O site combina uma narrativa histórica dividida em capítulos com um mini-game interativo que ensina o ciclo correto do manejo da terra.

---

## ❓ Para que serve?

O projeto serve como uma plataforma educacional interativa. Ele transforma o aprendizado sobre o agronegócio e a sustentabilidade em uma experiência visual e prática (gamificação). O objetivo é conscientizar o público jovem sobre como as ações corretas no campo garantem a preservação ambiental e a fartura na colheita.

---

## ⚙️ Como funciona?

A aplicação é dividida em três pilares principais integrados via código:

* **História em Capítulos**: Uma narrativa estruturada (Capítulos de 1 a 6) que guia o usuário pela evolução e os desafios da agricultura rústica e moderna.
* **Mini-game com Trava Lógica**: Um sistema de validação em tempo real. O jogador só consegue avançar se seguir a ordem agronômica real e sustentável do solo. Se clicar fora de ordem, a ação é bloqueada:
  1. **Arar** ➔ Prepara o solo e abre espaço para o oxigênio.
  2. **Adubar** ➔ Enriquece a terra com nutrientes essenciais.
  3. **Plantar** ➔ Insere as sementes no solo preparado.
  4. **Regar** ➔ Fornece a água necessária para a vida germinar.
* **Interatividade de Cenários**: O usuário pode alterar o plano de fundo do site (como Floresta ou Campo) através de botões dinâmicos, melhorando a imersão visual da experiência.

---

## 🎮 Como usar?

1. **Acesse o site oficial**: Abra o link do projeto [https://github.io](https://github.io).
2. **Navegue pelos capítulos**: Use os botões de controle para avançar ou voltar nas páginas da história.
3. **Customize o ambiente**: Utilize os botões de alteração de fundo para escolher o cenário visual que preferir.
4. **Jogue o mini-game**: Clique nos botões de ação respeitando estritamente a sequência correta da agricultura: **Arar ➔ Adubar ➔ Plantar ➔ Regar**.

---

## 🌱 Benefícios da Sustentabilidade no Campo

* **Preservação do Solo**: Evita a erosão e mantém a terra produtiva para o futuro.
* **Uso Consciente da Água**: Sistemas de rega precisa evitam o desperdício de recursos.
* **Equilíbrio Ecológico**: Respeita a fauna, a flora e os pequenos ecossistemas rurais.

---

## 🛠️ Tecnologias Utilizadas

* **HTML**: Estruturação dos textos históricos, painéis de controle e elementos interativos.
* **CSS**: Identidade visual responsiva adaptável para computadores e celulares.
* **JavaScript**: Lógica de navegação de páginas, troca de imagens de fundo e trava estrita do mini-game.
